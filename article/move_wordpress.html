Immagrating Wordpress to New Ubuntu Server

April 16, 2018 Wordpress Ubuntu

Before you start, make sure your Ubuntu server has the latest version of LAMP enviroment set up.

If you want install LAMP stack for your Ubuntu server, check here.

1. Install wordpress on your server.

CHECK HOW


2. Install the same theme and the same plugins.

It is important to install the theme and plugins before hands on the export-import so that we will have less errors.


3. Get export file.

Go to the origin wordpress site, find Tool > Export, choose "All content" Then click "Download Export File" to creat an "XML" file of your entire blog.


4. Using patched importer instead of WP importer if using beaver builder.

There seems to be an issue with the native WP importer. In Order to keep all the modules and same format, beaver builder has a patched importer to use: http://forum.wpbeaverbuilder.com/downloads/wordpress-importer-patched.zip

Download and install it into your destination wordpress site.


5. Importing.

Find Tool section of the Dashboard and select "Import", choose wordpress and upload the XML file you got in step 3. Remember to select "Download & import file attachments" to move your media along with the rest of your blog


6. Taking care of the rest.

We almost finish, now we have all the contents but still need set the front page to be the good one, change all the dependency link addresses to the right page. Making sure all the plugins are set up, check everything and you are good to go!
